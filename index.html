Semantic elements

Lesson 1 | HTML5100 | HTML and CSS Foundations

By Mike Watters

Semantic elements are meaningful tags in the code that help browsers and developers understand the purpose of what's contained within them. They differ from non-semantic tags like <div> and <span>, which are more generic tags that say nothing about the content between them.

Some examples of semantic tags include:

  • <article>
  • <navigation>
  • <summary>
  • <address>
  • <footer>

Using correct semantic elements helps search engines identify the correct content on the webpage. Other benefits include accessibility, internationalization, and interoptibility.Semantic elements help avoid "divitis," which is when web developers overuse <div> tags.
